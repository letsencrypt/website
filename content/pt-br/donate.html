Estamos tornando possível que todos possam experimentar uma Web segura e que respeita a privacidade. Nós tornamos fácil obter certificados HTTPS, porque a facilidade de uso é crucial para adoção. Provemos certificados gratuitos, porque o custo exclui pessoas. Nossos certificados estão disponíveis em todos os países do mundo, porque uma Web segura é para todos. Nós lutamos para sermos abertos e transparentes, porque estes valores são fundamentais para a existência de confiança.

Podemos receber doações via Bitcoin equivalentes a $1,000 USD ou mais. Por favor envie um e-mail para donate@letsencrypt.org para faturamento.
